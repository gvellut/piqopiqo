The "Undo ..." menu item is changed into "Redo ..." after the Undo has been called, as long as a new Undo session has not started using the criteria below.

The undo entries are discarded / new entry starts : when 1. a new selection is made (that is at least one additional item is selected or an existing selection discarded) AND 2. an edit is made. 
for example if the label is changed (Undo label entry is created). Then a new selection is made : step 2 (from above) not yet made : the Undo label entry is still active and can be called.
